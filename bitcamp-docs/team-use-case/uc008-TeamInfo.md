## 시나리오(Flow of Evnets)

### 기본 흐름(Basic Flows)

- 1. 액터가 팀 정보 버튼을 클릭했을 때 이 유스케이스를 시작한다.
- 2. 시스템은 팀 정보 폼을 출력한다.
- 3. 액터는 폼에서 종목, 지역을 선택하거나 팀명을 입력한 후 검색하기 버튼을 클릭한다.
- 4. 시스템은 액터의 검색 결과에 맞춰 팀 목록을 출력한다.
- 5. 액터는 목록중에 하나를 클릭한다.
- 6. 시스템은 팀 정보를 출력한다.
